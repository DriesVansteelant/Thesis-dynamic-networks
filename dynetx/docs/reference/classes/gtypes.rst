***********
Graph types
***********

DyNetx provides data structures and methods for storing graphs.

The choice of graph class depends on the structure of the
graph you want to represent.

Which graph class should I use?
===============================

==================  ========================
Dynamic Graph Type  DyNetx Class
==================  ========================
Undirected          DynGraph
Directed            DynDiGraph
==================  ========================
